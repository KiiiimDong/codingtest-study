# [최솟값찾기](https://www.acmicpc.net/problem/11003)
---
## 문제 요약
N개의 수 A1, A2, ..., AN과 L이 주어진다.

Di = Ai-L+1 ~ Ai 중의 최솟값이라고 할 때, D에 저장된 수를 출력하는 프로그램을 작성하시오. 이때, i ≤ 0 인 Ai는 무시하고 D를 구해야 한다.

## 문제유형
슬라이딩윈도우
